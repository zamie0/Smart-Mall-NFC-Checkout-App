# Smart Mall NFC Checkout App

A modern, intelligent shopping application that leverages NFC technology for seamless checkout experiences in smart malls. This app provides users with features like real-time price comparison, budget tracking, personalized deals, and offline synchronization.

## Features

- **NFC Scanning**: Quick product scanning and addition to cart
- **Smart Cart Management**: Add, remove, and manage items with ease
- **Price Comparison**: Real-time price comparison across stores
- **Budget Tracking**: Set and monitor shopping budgets
- **Personalized Deals**: AI-powered deal recommendations
- **Offline Sync**: Seamless synchronization when connectivity is restored
- **Receipt Management**: Digital receipts and transaction history
- **Shopping Lists**: Create and manage shopping lists
- **Nutrition Insights**: Nutritional information for products
- **Refill Reminders**: Automatic reminders for frequently purchased items

## Technologies Used

This project is built with:

- **Vite**: Fast build tool and development server
- **TypeScript**: Type-safe JavaScript
- **React**: Modern UI library
- **shadcn-ui**: Beautiful and accessible UI components
- **Tailwind CSS**: Utility-first CSS framework

## Installation

To run this project locally, ensure you have Node.js and npm installed. [Install Node.js with nvm](https://github.com/nvm-sh/nvm#installing-and-updating) if needed.

Follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd smart-mall-nfc-checkout-app

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

The application will be available at `http://localhost:5173` (or the port specified by Vite).

## Usage

1. **Authentication**: Log in or create an account
2. **Browse Products**: Explore available products with detailed information
3. **NFC Scanning**: Use NFC-enabled devices to scan products
4. **Manage Cart**: Add items, apply discounts, and proceed to checkout
5. **Smart Features**: Utilize budget tracking, deal recommendations, and insights
6. **Checkout**: Complete purchases with various payment methods
7. **Receipts**: View and manage digital receipts
